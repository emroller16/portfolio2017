Web Design

Core Lab: A UConn Wordpress Site

Professor Leighton Core needed a simple site to advertise his MCB lab to the UConn community. He was looking for a site to showcase the lab's cutting edge research as well as attract prospective student researchers. Using UConn's instance of Wordpress, I created a custom site to achieve the lab's goals.

Process

  • Brainstorming with Dr. Core to establish needs/goals for the site
  • Create wireframes to get feedback with the client
  • Create site in UConn's instance of Aurora with custom theming and CSS
  • Revise with client and publish for final review
  • Update site as needed

Outcomes

  • Final site with a distinguished feel
  • Prominent contact information for student researchers to apply
