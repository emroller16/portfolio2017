Web Design

Academic IT
A UConn Wordpress Site

To display a variety of IT solutions and offerings in a user friendly way, we needed a solution to group offerings in an intuitive way as well as offer a robust search tool to allow UConn professors to find the right tool for their needs. Additionally, we wanted to encourage exploration and discovery of new tools available.

Organizing

  • We first surveyed the full range of technologies available to professors, as well as existing UConn sites or pages with related support, request forms, and information.
  • Our team categorized the technologies into intuitive categories.
  • I sketched several options for how to display categories intuitively while also minimizing number of clicks to get to relevant information.
  • We finally decided on a grid system, with links to a specific page for each category.
  • Clean design with close attention to formatting to feature largely text-based content.

Outcomes

  • A grid layout displaying categories of IT offerings for users interested in exploring all offerings in a specific category or for users familiar with a the specific tool they're seeking.
  • A robust search function for users who aren't sure of the specific tool they're searching for.
  • An additional page listing all services offered for users who want to explore all available options.
  • Easy client maintenance.
  • Simple, straightforward navigation.
