Web Design

Software.uconn.edu
A UConn Wordpress Site

UConn's software catalog needed a new look and feel. Our team at UConn's Web Development lab aimed to create a catolog that was intuitive to use and made it easy to find specific software available. We needed a way to organize software available to students vs available to faculty/staff vs free/open source software available to anyone. Additionally, we needed a way to portray which software offerings could be accessed from computer labs on campus or from UConn's virtual server.

Iterative Design Process

  • We created several iterations of how to catalog and organize software offerings to best present ----------
  • To track software download data, we created forms that served to both provide download information to users as well as record download numbers for each software.
  • We embedded maps of computer labs for each campus as well as created a key for specific types of software and printing options provided at each.
  • For each software offering, we created consistent pages with key information about accessibility, --------------

Outcomes

  • An average of XX downloads per month.
  • A central site to find computer labs and their offerings across each of UConn's branch campuses.
  • Software offerings organized by student downloads, faculty downloads, and free/open source programs.
  • A robust search tool to find specific software offerings.
  • A consistent key to indicate which software is available through UConn's virtual servers.
